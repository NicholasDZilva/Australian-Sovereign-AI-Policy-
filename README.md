## Overview

This repository contains a draft policy framework for establishing sovereign control over artificial intelligence (AI) systems in Australia. It addresses:

- Energy architecture and planning for AI infrastructure
- Sovereignty through control of the AI framing layer
- Structural oversight and independent Sentinel Layer monitoring
- Recommendations for maintaining continuous, verifiable AI sovereignty

The document is intended for policymakers, government agencies, and industry stakeholders.
